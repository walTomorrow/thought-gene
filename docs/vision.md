# Vision

Thought Gene exists to solve a growing problem in AI-assisted work:

Conversations are becoming easier.

Remembering what was learned from them is not.

As projects become increasingly collaborative between humans and AI systems, knowledge is scattered across chats, documents, issues, notes, and code repositories. Decisions become buried inside conversation histories. Deferred work gets forgotten. Open questions disappear. New agents repeatedly require the same context to be explained again.

Thought Gene aims to become a persistent memory layer for project reasoning.

## Conversations Are Temporary

Most AI tools are built around conversations.

Thought Gene is built around what survives conversations:

* Decisions
* Questions
* Assumptions
* Risks
* Constraints
* Plans
* Context

The conversation is not the end product.

The knowledge produced by the conversation is.

## From Chats to Knowledge

A project should not be a collection of chat transcripts.

A project should be a living body of structured knowledge.

Every important discussion should eventually produce durable artifacts:

* Accepted decisions
* Rejected alternatives
* Deferred implementations
* Open questions
* Rationale
* Consequences

These artifacts should remain accessible long after the original conversation has ended.

## Branches as Investigations

Questions naturally branch.

A project conversation may generate dozens or hundreds of decisions over time.

Thought Gene treats branches as temporary investigations rather than permanent containers.

Branches exist to answer questions.

Once useful conclusions have been reached, the branch can be closed.

The knowledge remains.

The branch becomes history.

## Project Memory

The project itself should develop memory.

Not memory in the form of raw conversation history, but memory in the form of structured understanding.

The system should always be able to answer:

* What have we decided?
* Why did we decide it?
* What assumptions are we making?
* What remains unresolved?
* What work has been intentionally postponed?
* What changed since the last major milestone?

## AI-Native Knowledge

Future projects will involve many AI systems.

Different models will assist with:

* Planning
* Research
* Architecture
* Coding
* Documentation
* Analysis

Thought Gene aims to become the shared memory layer between those systems.

Every agent should be able to understand the current state of a project without requiring its entire history.

## Long-Term Goal

The long-term goal is simple:

Convert temporary conversations into durable project knowledge.

Projects should accumulate understanding, not transcripts.

Reasoning should become an asset rather than a forgotten byproduct of discussion.
